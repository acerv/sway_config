# Installation

Install dependences first:
```
sudo zypper in \
    sway \
    SwayNotificationCenter \
    swaylock \
    swayidle \
    waybar \
    kitty \
    grim \
    wl-clipboard
```

Then copy configurations:
```
cp -r \
    sway \
    waybar \
    kitty \
    wofi ~/.config
```
